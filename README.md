# CityExplorer
 Ingeniería de Software II Enero - Abril 2025


El proyecto consiste en desarrollar una emocionante aplicación web que será una guía completa para turistas en San Miguel de Allende. El sitio ofrecerá una amplia gama de contenidos, incluyendo categorías para descubrir los mejores lugares para relajarse, salir de fiesta, disfrutar de bebidas, o simplemente deleitarse con una deliciosa comida. También presentará las mejores rutas para pasear por las calles con vistas impresionantes del centro de la ciudad, así como lugares históricos y atracciones interesantes. 
 
Además, la aplicación contará con un espacio dedicado a comentarios, valoraciones, sugerencias y recomendaciones de nuevos lugares que aún no estén en la plataforma. El administrador tendrá la capacidad de añadir o eliminar nuevos destinos, así como de eliminar comentarios ofensivos o contenido inapropiado y la opción de guardar los lugares favoritos para visitar más tarde.
